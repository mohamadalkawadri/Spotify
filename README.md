# Spotify Listening History Analysis

This project analyzes Spotify streaming history data using Python and Pandas.

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib

## Features
- Data cleaning and preprocessing
- DateTime handling
- Artist and song popularity analysis
- Listening behavior by hour of day
- Skip-rate analysis
- Visualization of listening patterns
- Ariana Grande listening habit deep dive

## Learning Outcomes
- DataFrame manipulation
- Exploratory Data Analysis (EDA)
- Data visualization
- Grouping and aggregation with Pandas
- Working with time-series data
